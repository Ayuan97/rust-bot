# 截图资源目录

请在此目录放置以下截图文件：

## 必需文件

| 文件名 | 说明 | 建议尺寸 |
|--------|------|----------|
| `logo.png` | 项目 Logo | 120x120 px |
| `screenshot-dashboard.png` | 主界面截图 | 1280x720 px |

## 可选文件

| 文件名 | 说明 | 建议尺寸 |
|--------|------|----------|
| `screenshot-server.png` | 服务器信息面板 | 400x300 px |
| `screenshot-chat.png` | 队伍聊天面板 | 400x300 px |
| `screenshot-devices.png` | 设备控制面板 | 400x300 px |

## 截图建议

1. 使用浏览器开发者工具截图，确保清晰度
2. 隐藏敏感信息（服务器 IP、玩家名等）
3. 使用 PNG 格式保持质量
4. 可使用工具压缩图片大小（如 TinyPNG）
